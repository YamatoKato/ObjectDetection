**1. 制作物の動作概要**

読み込んだ画像ファイルからAzureのComputer VisionAPIを用いてオブジェクト(物体)の検出を行うアプリ。


**2.使用した技術**

・Streamlit
・AzureのComputer VisionAPI
・Python3.x
